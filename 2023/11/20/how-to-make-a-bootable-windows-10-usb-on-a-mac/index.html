How to make a bootable windows 10 USB on a Mac

How to make a bootable windows 10 USB on a Mac

Inset the USB driver

Open the disk utility and format the USB driver

Download the windows 10 ISO file

https://www.microsoft.com/en-us/software-download/windows10ISO

Step-0: Double click the ISO file to mount the disk

Step-1: Copy all the files expect the folder sources from ISO mounted driver to USB driver

Step-2: Create a new folder sources in USB driver

Step-3: Copy all the files except the install.wim under sources folder of ISO mounted driver, into new folder sources in USB driver

Split the big file: install.wim

Brew install wimlib

Split the big file: install.wim and output the splited files to sources folder of USB driver
